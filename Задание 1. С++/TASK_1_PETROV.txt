Чтение файла можно осуществлять построчно конструкцией с оператором
for, читая в цикле и увеличивая счетчик для каждой строки в файле.
>>> fhand = open('mbox.txt')
>>> count = 0
>>> for line in fhand:
		count = count + 1
>>> print('Line Count:', count)
Line Count: 132045
Причина, по которой функция open не читает весь файл, заключается в его
большом объёме (мега- и гигабайты). Функция open занимает одинаковое
количество времени, независимо от размера файла.
В случае, когда файл читается с использованием цикла for, Python заботится
о разбиении данных в файле на отдельные строки, используя символ новой
строки. Python читает каждую строку, используя новую строку, и включает новую
строку, как последний символ в переменную line для каждой итерации цикла for.
Если известно, что файл относительно небольшой по сравнению с размером
оперативной памяти, то можно прочитать весь файл в одну строку, используя
метод read для дескриптора файла.
>>> fhand = open('mbox-short.txt')
>>> inp = fhand.read()
>>> print(len(inp))
94626
>>> print(inp[:20])
From stephen.marquar
В примере содержимое файла mbox-short.txt (94626 символов) прочитано
в переменную inp. Затем взят срез из 20 символов по строке в inp.
